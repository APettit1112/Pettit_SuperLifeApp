# Super Campus Life

## Description
A responsive website showing campus events, dining info, campus calendar with events. Built with semantic HTML, Bootstrap, custom CSS, and JavaScript. Demonstrates external API integration and accessibility.

## Technologies
- HTML5 (semantic)
- CSS3 (custom + media queries)
- Bootstrap 5.3.2
- JavaScript (ES6)
- External API: Open-Meteo

## Pages
- `index.html` — Home + announcements + weather widget
- `dining.html` — Dining menu + hours (menu can be refreshed via JS)
- `calendar.html` — Events listing
